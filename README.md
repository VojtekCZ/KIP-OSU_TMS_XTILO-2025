# KIP-OSU_TMS_XTILO-2025
Simulátor deterministického k-páskového Turingova stroje v Pythonu.

Tento projekt implementuje simulátor deterministického k-páskového Turingova stroje. Umožňuje definovat konečnou množinu stavů, počáteční a koncové stavy, páskovou abecedu (včetně prázdného symbolu ε), přechodové funkce a poskytuje serializované zakódování stroje. Simulátor tiskne stav pásek v každém kroku a na konci vypíše kód stroje.
